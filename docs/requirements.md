# 新規事業 要件定義

## プロジェクト概要
エンタメ × テック領域で、若年女性をターゲットにしたバイラル成長するサービスを企画する。

## 前提条件

| 項目 | 内容 |
|------|------|
| ターゲット | 高校生女子〜25歳くらいの女性 |
| 成長戦略 | マーケティング（口コミ/バイラル）で伸びる |
| 市場特性 | ニッチだけどスケーラブル |
| 利用シーン | 少人数（1人〜数人）でも楽しめる |
| 領域 | エンタメ × テック |
| IP展開 | キャラクターIPに興味あり |
| 創業者の関与 | 開発者/創業者として裏方 |
| 収益モデル | 未定（まずユーザー獲得優先） |

## 成功のベンチマーク：BeRealから学ぶ3法則

### 1. 制約が面白さを生む
- 1日1回、ランダム通知、2分以内の撮影
- **「制限があるからこそハマる」** 仕組み
- SNS疲れの逆張りとしてZ世代に刺さった

### 2. 少人数で成立する
- 友達数人でも十分楽しめる設計
- ネットワーク効果なしでも個人で価値がある
- 大量ユーザーがいなくても初期から楽しい

### 3. プロダクト外でバイラルする
- TikTokで「BeRealの時間だ！」がミーム化
- **サービスの"結果"がコンテンツになる** 設計
- ユーザーが勝手に広めてくれる

## 参考にすべきサービス
- **BeReal** — 制約×日常×バイラル
- **Zenly** — 位置情報×友達×少人数
- **たまごっち** — キャラ育成×愛着×IP展開
- **LINEスタンプ** — キャラクターIP×日常利用
- **Character.AI** — AIキャラとの対話体験
