# 新規事業アイデア 5案比較サマリー

## 前提条件（共通）
- **ターゲット**: 高校生女子〜25歳くらいの女性
- **成長戦略**: マーケ（口コミ/バイラル）で伸びる
- **市場**: ニッチだけどスケーラブル
- **利用**: 少人数（1人〜数人）でも楽しめる
- **領域**: エンタメ × テック × キャラクターIP

## 各案の強み・弱み要約

### 案A: うちの子（合計28点 — 最高評価）
- **強み**: 1人で完全に成立、開発コスト低い、「うちの子」文化でバイラル、IP展開しやすい
- **弱み**: Character.AIとの差別化が必要、LLM APIコスト管理
- **一言**: 全条件を最もバランスよく満たす。MVPも最速で出せる

### 案B: ジャッジちゃん（合計22点）
- **強み**: 友達との自然な遊び、実況キャラの面白さ、招待導線が自然
- **弱み**: グループ作成ハードル、お題ネタ切れ、1人では楽しめない
- **一言**: 友達軸は強いが、初期ユーザー獲得のハードルあり

### 案C: まちキャラ（合計21点）
- **強み**: 位置情報×キャラの新しさ、ローカルIP展開、商業施設コラボ
- **弱み**: 開発コスト最大、プライバシー問題、地方格差
- **一言**: ポテンシャル高いが、MVPが重すぎる。資金調達後なら

### 案D: 運命のコトバ（合計25点）
- **強み**: 開発コスト最小、占い×女性は鉄板、朝のルーティン化
- **弱み**: 占いアプリは飽和市場、差別化がデザイン依存
- **一言**: 最もローリスク。ただしレッドオーシャンとの戦い

### 案E: うちの寮（合計23点）
- **強み**: かなりニッチでハマる層が深い、二次創作の可能性、受動的に楽しめる
- **弱み**: ストーリーエンジン開発が重い、品質がサービスの生命線
- **一言**: 刺さる層にはめちゃくちゃ刺さる。ストーリー品質次第

## PM所感

### 推しランキング
1. **案A: うちの子** — 全条件に対してバランス最強。MVPも早い。まず出して検証できる
2. **案D: 運命のコトバ** — ローリスク・ローコストで始められる。ただし差別化が課題
3. **案E: うちの寮** — 一番ニッチでユニーク。刺さったら最強だがリスクも高い
4. **案B: ジャッジちゃん** — 友達との遊びとしては面白い。B+Aの要素ミックスも面白そう
5. **案C: まちキャラ** — 可能性は大きいが、初期投資が重すぎる

### 組み合わせの可能性
- **A+E**: 「うちの子」を複数人で寮に住ませる → 1人でも楽しい + 友達いるとさらに楽しい
- **A+D**: キャラが毎朝占いも届けてくれる → 機能追加としてシンプル
- **B+A**: お題バトルにそれぞれの「うちの子」が参加 → 友達要素の追加
