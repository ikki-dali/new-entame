# 案D: 1日1回キャラ占い「運命のコトバ」

## 一言で言うと
> 毎朝、あなた専属のAI占い師キャラが「今日の運命のコトバ」を届けてくれる。1日1回だけ質問もできる。回答は毎回違う美しいカードデザインで、シェアしたくなる。

## コンセプト

### 体験の流れ
1. 初回起動 → 占い師キャラを選ぶ（タロット系、星座系、ギャル霊媒師系など個性的）
2. 毎朝通知：「今日のコトバが届いたよ」
3. カードを開くと、**今日のあなただけの「運命のコトバ」**が表示
4. 1日1回だけ占い師に質問できる（恋愛、仕事、友達関係など）
5. 回答は**美麗なカードデザイン**で生成される
6. カードをInstagramストーリーやLINEにシェア

### 占い師キャラの例
- **月読（つくよみ）**: 神秘的な和風占い師。言葉が詩的
- **ギャルミ**: ギャル霊媒師。ノリは軽いけど核心をつく
- **Dr.ステラ**: 宇宙っぽい理系占い師。データで未来を語る
- **ねこまる**: 猫の占い師。ゆるいけど時々鋭い

## ターゲットに刺さる理由

### 占い×女性は鉄板
- 10代〜20代前半女性の占い利用率は極めて高い
- 朝の占いチェックは既に習慣になっている人が多い
- **既存の行動（朝の占いチェック）の上位互換を提供**

### 「毎朝の小さな幸せ」
- 開くのが楽しみ = DAU（日次アクティブユーザー）が高くなる
- 短時間で完結（30秒〜1分）
- **朝のルーティンに組み込まれやすい**

### カードのシェア文化
- Instagramストーリーに占い結果を載せるのは既に定番
- **「映える占いカード」はシェアのハードルが低い**
- 「私もこのアプリの占い毎日見てる！」の会話が生まれる

## バイラル設計

### 自然拡散ポイント
1. **Instagramストーリー**: 美しいカードデザイン → 毎朝シェアする人が出現
2. **「今日のコトバ」文化**: 友達同士で「今日なんだった？」の会話
3. **キャラの言い回し**: 占い師キャラの名言がスクショで拡散
4. **カップル/友達おそろい**: 同じ占い師を選んでる友達同士の文化

### インフルエンサー活用
- 占い系インフルエンサーとの相性が良い
- 「このアプリの占い、ガチで当たる」系のレビュー
- 占い師キャラのコラボ（有名占い師監修キャラ）

## 少人数での楽しみ方
- **1人**: メイン体験。毎朝のルーティン
- **2人**: 「今日のコトバ」を見せ合う。カップルで相性占い
- **グループ**: 友達の今日の運勢ランキング

## IP展開の可能性
| 展開 | 内容 |
|------|------|
| キャラグッズ | 占い師キャラのアクスタ、タロットカード |
| コラボ占い | 雑誌の占いコーナー監修 |
| リアルイベント | 占いフェスにブース出展 |
| 書籍 | 「運命のコトバ」書籍化 |

## 技術要件（概算）
- **LLM API**: 占い文章生成（キャラの口調付き）
- **画像生成**: カードデザイン自動生成
- **バックエンド**: ユーザーデータ、占い履歴管理
- **フロントエンド**: iOS/Androidアプリ
- **MVP開発期間**: 2-3ヶ月（最もシンプル）

## リスク
- 占いアプリは飽和市場（レッドオーシャン）
- 差別化がカードデザインとキャラに依存
- 占いの「当たった」感をAIで出せるか
- マネタイズが難しい（占いアプリの課金率は低い傾向）
