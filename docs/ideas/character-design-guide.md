# キャラ設計ガイドライン

## いっきさんの思想（ブレストで出た核心）

### 最上位コンセプト
> **「このキャラ、本当にこの世界に住んでるんじゃない？」**

プロダクトもマーケも、全てこの1行に集約される。

### コア原則
1. **いかにユーザーとキャラの接触回数を"無意識に"増やすか** — 能動的にアプリを開くんじゃなくて、日常のあちこちに「あ、またいる」が発生する
2. **たまごっち寄り** — ポケモン（コレクション・バトル）ではなく、お世話・愛着・自分だけの子が育つ体験
3. **大人が幼いものを楽しんじゃってる** — 「赤ちゃんが楽しめるようなものを女子高生の私が楽しんでる」が成立する時代
4. **キャラが勝手に動く** — ユーザーが操作するんじゃなく、キャラが自律的に生活してる

### マーケの方向性
> **過激だけど誰もやってない。広告の顔をしてない。**

街にキャラの生活の痕跡を残す。発見する楽しさがある。

## 成功IPから学んだキャラ設計の法則

### 法則1: 「かわいい × 予想外の何か」のギャップが必要
かわいいだけじゃ差別化できない。成功IPは全て「ズラし」を持ってる。

| IP | かわいい × | ギャップ |
|---|---|---|
| ちいかわ | かわいい見た目 | × 残酷で理不尽なストーリー |
| すみっコぐらし | ゆるいデザイン | × 全員がネガティブ属性（弱さの肯定） |
| モルカー | かわいいフェルト | × 社会風刺 |
| Jellycat | ぬいぐるみ | × ラグジュアリー |
| Labubu | 妖精 | × キモかわいい（Ugly-Cute） |
| リラックマ | くまのキャラ | × 怠惰（ダラダラ） |

**→ このキャラの「ズラし」は何か？はまだ未定。無理に決めず、体験の中から自然に見つける。**

### 法則2: 感情を投影できる「余白」を残す
- キティの口がない → 見る人が勝手に表情を読み取る → 一人ひとりにとって「自分だけの関係」が生まれる
- 完璧にデザインしすぎない。ユーザーが感情を乗せられる余白があるキャラが長寿IPになる

### 法則3: 弱さ・不完全さが共感を生む
- 成功IPのキャラは全員「完璧ではない」
- 弱く、不器用で、時に理不尽な目に遭う
- SNS疲れ・競争社会のストレスを抱える世代にとって「弱くてもいい」は強烈なメッセージ
- **「強いキャラ」より「ダメなキャラ」の方が愛される時代**

### 法則4: コレクタブル性（集めたくなる設計）
- Squishmallows: 3000種以上
- Labubu: レア版・限定版で転売市場形成
- すみっコぐらし: 複数キャラ展開で推しが選べる
- **1体で完結させない。バリエーション・仲間・限定版の余地を最初から設計に入れる**

### 法則5: 「推し活」構造を組み込む
- キャラを「推す」行動（グッズ収集、SNS布教、投票、コレクション）はアイドル推しと同じ構造
- サンリオキャラクター大賞、Squishmallowsのコレクション文化がこれ
- **ファンが「布教したくなる」設計**を最初から考える

## 0→1の3パターン（どのルートで行くか）

成功IPの最初のファン獲得には3つしかパターンがなかった。

### パターン1: 既存ファンから派生（ちいかわ型）
- 別のプラットフォームで先にファンベースを構築 → 新IPを派生
- ナガノ氏：LINEスタンプで33万フォロワー → ちいかわ専用アカウント開設
- **再現するなら：先にLINEスタンプやTikTokアカウントでキャラを育ててからプロダクト展開**

### パターン2: 物理で「偶然の出会い」を作る（すみっコ型）
- リアル店舗・ガチャ・クレーンゲームで手に取った人がファンに
- すみっコは文房具、Squishmallowsはクレーンゲームから
- **再現するなら：キーホルダーやガチャでまず物理的に存在させる**

### パターン3: プラットフォームの波に乗る（Pusheen型）
- SNSの新機能・アルゴリズムに最適なコンテンツだった
- PusheenはFacebook公式スタンプに採用されて爆発
- **再現するなら：TikTokやInstagramの新機能に最適化したコンテンツを出し続ける**

### いっきさんの方向性
上記に加えて**「ゲリラで一気に認知を取る」（Labubu型）**を組み合わせる。
有名人プレゼント、渋谷ジャック、学校突撃、フラッシュモブ等で初速をつける。

## 最も重要な示唆

> **「バズを狙う」より「火がつく準備を整えて、継続する」方が再現性が高い。**

全成功IPの初期は広告費ほぼゼロ。セレブが取り上げたのもTikTokでバズったのも全て結果であって計画ではない。
ただし「いつ火がついてもいいように準備ができている状態」は必須。
