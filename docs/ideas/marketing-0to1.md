# キャラ認知施策（0→1）

## 戦略の軸
> 「このキャラ、本当にこの世界に住んでるんじゃない？」と思わせる。広告ではなく、キャラの生活の痕跡を街に残す。

## 基本構造
リアルで「なにあれ？」を作る → 撮る → SNSに逆流する

## 共通する設計思想

1. **広告の顔をしていない** — 全て日常に紛れ込んでる
2. **発見する楽しさがある** — 見つけた人が「報告」したくなる
3. **キャラが生きてる感** — 人間がやることをキャラがやってる
4. **撮りたくなる** — 写真/動画映えする → SNSに自然に上がる
5. **ターゲットのいる場所に行く** — 待つんじゃなくて会いに行く

## 参考：成功IPの0→1の法則

- 全成功IP（ちいかわ、すみっコ、Squishmallows、Labubu）の初期は広告費ほぼゼロ
- 「バズを狙う」より「火がつく準備を整えて継続する」方が再現性が高い
- セレブ起点の爆発（Labubu×リサ）は計画ではなく結果だった
- ただし「準備ができている状態」は必須

## 未整理メモ

- ストーリーが大事。キャラには物語が必要
- ゲリラ活動自体がキャラのストーリーになる可能性
- 「キャラが本当にこの世界で生活してる」が全施策の軸
- プロダクト案（キーホルダー、代理SNS、写真写り込み、キャラ同士の関係）ともこの世界観で一貫する
