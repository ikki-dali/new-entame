# キャラ認知施策（0→1）

## 戦略の軸
> 「このキャラ、本当にこの世界に住んでるんじゃない？」と思わせる。広告ではなく、キャラの生活の痕跡を街に残す。

## 基本構造
リアルで「なにあれ？」を作る → 撮る → SNSに逆流する

---

| 施策 | 内容 |
|------|------|
| **有名人にプレゼント** | キャラグッズを贈る。Labubu×リサのように自発的に好きになってもらう設計 |

### ティア2: 街にキャラの「生活の痕跡」を残す
| 施策 | 内容 |
|------|------|
| **ベンチにキャラ置く** | 街中のベンチにフィギュアをゲリラ設置。発見した人が撮る。人気出たらプレミア化 |
| **指名手配ポスター** | 「この子を見かけたらDMください」脱走中の設定。目撃情報がSNSに集まる |
| **落とし物を置く** | 原宿にキャラのかわいいポーチ等を落とす。「これ誰の？」がSNSに上がる |
| **NFC町中に置きまくる** | 触ったらキャラが出てくる。「なにこれ？」が大量発生 |
| **雨の日だけ浮かび上がる** | 撥水加工で歩道に描く。晴れの日は見えない。「雨の日に渋谷行ってみて」 |
| **Googleマップにキャラのレビュー** | 「このカフェのプリンおいしかったです ーキャラ名」を本当に載せる |

## 共通する設計思想

1. **広告の顔をしていない** — 全て日常に紛れ込んでる
2. **発見する楽しさがある** — 見つけた人が「報告」したくなる
3. **キャラが生きてる感** — 人間がやることをキャラがやってる
4. **撮りたくなる** — 写真/動画映えする → SNSに自然に上がる

---

## 参考：成功事例から学んだ0→1の法則

- 全成功IP（ちいかわ、すみっコ、Squishmallows、Labubu）の初期は広告費ほぼゼロ
- 「バズを狙う」より「火がつく準備を整えて継続する」方が再現性が高い
- セレブ起点の爆発（Labubu×リサ）は計画ではなく結果だった
- ただし「準備ができている状態」は必須

## 未整理メモ

- ゲリラ活動自体がキャラのストーリーになる可能性
- 「キャラが本当にこの世界で生活してる」が全施策の軸
- プロダクト案（キーホルダー、代理SNS、写真写り込み、キャラ同士の関係）ともこの世界観で一貫する
